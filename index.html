This extension bridges the critical gap between identifying accessibility issues and fixing them. While tools like Lighthouse and axe-core excel at diagnostics, they leave users waiting for developers to implement fixes‚Äîa process that can take weeks, months, or may never happen.

Our user-centric approach provides immediate remediation, shifting from a compliance-driven diagnostic model to a user-empowering solution. The extension works universally across all websites with negligible performance impact.

Get Started in Minutes

Install the extension and start enhancing web accessibility today

1

Clone the Repository

Download the source code from GitHub

git clone https://github.com/FrankenSama/accessibility-text-enhancer.git
cd accessibility-text-enhancer
2

Enable Developer Mode

Navigate to chrome://extensions/ and toggle on "Developer mode" in the top-right corner

3

Load the Extension

Click "Load unpacked" and select the accessibility-text-enhancer directory

4

Start Enhancing!

Select any text on any webpage to see the accessibility toolbar appear. Click features to apply enhancements instantly.
