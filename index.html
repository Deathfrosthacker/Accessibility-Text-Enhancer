Capabilities

Eight Accessibility Tools

Select any text on any webpage to activate the toolbar and apply enhancements instantly.

B

Bold Toggle

Increase font weight instantly for better readability. Ideal for users with visual impairments or reading challenges.

H

Smart Highlighting

Apply yellow backgrounds or outlines to any selected text. Helps with focus and information retention.

A

Font Scaling

Dynamically resize text from 8px to 50px. More precise than browser zoom — targets only what you select.

C

Contrast Fixer

Automatically switch text to black or white based on background brightness to meet WCAG contrast guidelines.

S

Text Spacing

Adjust letter and word spacing to reduce visual crowding — particularly effective for dyslexia-friendly reading.

R

Read Aloud

Built-in text-to-speech using the Web Speech API. Works immediately in-browser, no extra software needed.

U

Undo / Redo

Full 50-state history with keyboard shortcuts Ctrl+Z and Ctrl+Y. Experiment freely and revert any change.

P

Preset Profiles

One-click profiles: Reading Mode, Dyslexia Friendly, High Contrast, and Focus Mode — saved per website.

Installation

Get Started in Minutes

Three ways to install — choose what works best for you.

1

Option A — Chrome Web Store (Recommended)

Visit the Chrome Web Store listing, click "Add to Chrome", and the extension installs automatically with auto-updates.

2

Option B — Manual Install (Developer Mode)

Download or clone the repository, open chrome://extensions/, enable Developer Mode, click "Load unpacked" and select the project folder.

git clone https://github.com/FrankenSama/accessibility-text-enhancer.git
3

Start Enhancing

Select any text on any webpage. The accessibility toolbar appears automatically at the top of the screen. Click any button to apply an enhancement instantly.

Research-Backed

Built on Academic Research

Developed as part of an MSc Computer Engineering dissertation at Solent University. Tested across 20 diverse websites including news sites, e-commerce platforms, government portals, and single-page apps.

The extension bridges the gap between identifying accessibility issues and actually fixing them — without waiting for developers.
